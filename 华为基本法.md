# 华为基本法

## 概述

《华为基本法》是华为的**企业宪法**，起草于1995—1998年，由任正非与中国人民大学彭剑锋教授等专家团队共同完成。它是中国第一部系统阐述企业价值观、战略思想与管理原则的纲领性文件。

> "资源是会枯竭的，唯有文化才会生生不息。" —— 任正非

## 核心内容

### 企业的基本假设

| 方面 | 表述 |
|------|------|
| 追求 | 在电子信息领域实现顾客的梦想 |
| 员工 | 认真负责和管理有效的员工是华为最大的财富 |
| 技术 | 广泛吸收世界电子信息领域的最新研究成果 |
| 精神 | 爱祖国、爱人民、爱事业、爱生活 |

### 核心价值观（基本法时期）

1. **以人为本** — 尊重个性，集体奋斗
2. **开放合作** — 在独立自主基础上开放合作
3. **自我批判** — 在自我批判中进步
4. **从摸爬滚打中总结** — 从实践中来，到实践中去

## 历史意义

- **制度起点**：标志着华为从人治走向法治
- **文化里程碑**：首次将华为文化系统化、文本化
- **企业宪法**：为后续管理变革（[[IPD流程体系]]、ISC等）奠定制度基础
- **时代印记**：中国民营企业走向规范化的标志性事件

## 局限与演进

1998年定稿后，任正非本人也承认：

> "《华为基本法》完成了它的历史使命，华为管理已经超越了基本法的框架，进入更深入的流程变革阶段。"

2008年后，《[[华为人力资源管理纲要]]》2.0逐步取代了基本法的主体地位。
