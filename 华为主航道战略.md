# 华为主航道战略

## 概述

**主航道战略**是华为最核心的战略原则：**聚焦于信息与通信技术（ICT）的主航道，不在非战略机会点上消耗战略竞争力量。**

> "力出一孔，利出一孔。当台风来的时候，猪都能飞起来，但华为要做的是把力量集中在主航道上。" —— 任正非

## 核心原则

### 力出一孔

- 资源高度集中于主航道
- 不在非战略市场分散精力
- 拒绝多元化的诱惑

### 利出一孔

- 收益只能来自为客户创造价值
- 禁止员工通过关联交易、外部投资获取不当利益
- 保证组织的纯洁性

## 什么是华为的"主航道"

| 时代 | 主航道定义 |
|------|------------|
| 1990s | 通信网络设备（交换机、传输） |
| 2000s | ICT基础设施（无线、光、数通） |
| 2010s | 联接 + 云计算 + 终端 |
| 2020s | 智能世界（联接+计算+AI） |

## 聚焦的代价与坚持

- **不搞房地产**：2000年深圳房地产最热时，任正非坚决不碰
- **不做互联网**：不追求风口上的机会
- **不搞多元化**：拒绝进入与主业无关的领域

> "在大机会时代，千万不要机会主义。" —— 任正非 2014

## 主航道与"备胎"

主航道战略并不排斥"备胎"准备：

- [[海思半导体]] 是主航道的"压舱石"，而非多元化
- 在主航道内部做多路径探索，但不偏离主线
- [[鸿蒙操作系统]] 服务于ICT生态延续
