# 自我批判文化

## 概述

**自我批判**是华为文化的核心机制之一，也是任正非最强调的组织能力。它不是个人层面的自我检讨，而是**组织层面的纠偏机制和进化引擎**。

> "自我批判是华为不断进步的工具。如果不进行自我批判，华为的衰落可能在第一时间就会到来。" —— 任正非

## 为什么要自我批判

1. **克服成功惯性** — 过去的成功是未来最大的障碍
2. **保持组织活力** — 对抗 [[熵减与华为管理]] 中的组织熵增
3. **防止官僚化** — 警醒大企业病、部门墙
4. **持续改进** — 流程优化、管理进步的驱动力

## 运作机制

| 机制 | 说明 |
|------|------|
| 民主生活会 | 管理层定期开展批评与自我批评 |
| 蓝军部 | 专门设置唱反调的部门，模拟竞争对手攻击 |
| 心声社区 | 员工实名/匿名发表批评意见，高层必须回应 |
| 红蓝军对抗 | 针对重大决策进行正反方辩论 |

### 蓝军机制

华为设有专职的"蓝军部"，其作用是：

> "蓝军想尽办法否定红军（正方案），这是自我批判的极致形式。"

详见 [[红蓝军对抗机制]]

## 历史事件

- **2000年**：任正非发表《华为的冬天》，拉开全员自我批判序幕
- **2008年**：《从泥坑里爬起来的人就是圣人》讲话
- **2014年**：大规模反腐运动，自我批判制度化
- **2018年**：人力资源管理纲要2.0，将自我批判写入制度
