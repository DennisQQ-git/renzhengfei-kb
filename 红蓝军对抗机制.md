# 红蓝军对抗机制

## 概述

**红蓝军对抗机制**是华为将[[自我批判文化]]制度化的核心设计。红军代表正方案或现行策略，蓝军代表挑战者——**专门负责"唱反调"**，从不同角度发起攻击，以暴露方案的漏洞和风险。

> "蓝军想尽办法否定红军，这是自我批判的极致形式。" —— 任正非

## 组织设置

- 华为设有正式编制的 **"蓝军部"**
- 蓝军成员专门负责挑刺、质疑、模拟竞争对手
- 蓝军的考核不来自红军认可，而是来自**发现问题的质量**

## 运作方式

| 维度 | 说明 |
|------|------|
| 战略对抗 | 对重大战略决策进行沙盘推演 |
| 产品攻防 | 模拟竞争对手攻击华为产品 |
| 流程审计 | 从使用者角度质疑流程合理性 |
| 决策挑战 | 在决策层提供反向意见报告 |

## 经典案例

### 备胎计划

蓝军早在2012年前后就预警了单一供应源的风险，推动了[[海思半导体]]"备胎"战略的加速。

### 终端战略

在华为要不要做自有品牌手机的问题上，蓝军与红军进行了长达数年的激烈辩论，最终推动了消费者BG的战略升级。

## 成功要素

1. **独立性** — 蓝军不受红军管辖
2. **专业性** — 蓝军成员需具备行业顶尖水平
3. **结果导向** — 以发现问题的质量而非数量考核
4. **高层支持** — 任正非亲自为蓝军背书
